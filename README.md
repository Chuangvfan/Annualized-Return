# 基金/股票年化收益记账本 
## 核心功能

1.  **真实年化计算**
    本工具采用 XIRR 模型，计算投资组合的年化回报。

2.  **资金流向管理**
    *   **期初设定**：锁定账户的起始资金与日期作为计算基准。
    *   **交易记录**：支持记录“买入（投入）”与“卖出（取出）”操作。
    *   **视图分组**：交易明细自动按月份归类折叠。
    *   **定投管理**：增加定投功能

3.  **期末结算**
    只需输入当前的账户总市值，系统即可结合历史流水，一键计算出当前的年化收益率及绝对盈亏金额。

4.  **数据本地化**
    所有数据以 JSON 格式存储于本地，支持一键导出备份与恢复。
